# agent-awareness-plugin-quota

Provider-aware quota visibility for Claude Code and Codex.

## Install

```bash
npm install -g agent-awareness-plugin-quota
```

## Config

`~/.config/agent-awareness/plugins.d/quota.json`

```json
{
  "showSession": true,
  "showBurst": true,
  "showWeekly": true,
  "showResetTime": true
}
```
